# A Basic Chess Engine Written in C++

## Features
### Algorithms:
    * Negamax
    * Alpha Beta Pruning
    * Move ordering

### Other:
    * A UI for playing against the engine or for watching two engines playing

## Future goals
    * Bunch of Algorithms
    * Designated UI for engine vs engine


## Libaries this engine is using
    * SFML for graphics
    * chess-libary by Disservin (https://github.com/Disservin/chess-library)
